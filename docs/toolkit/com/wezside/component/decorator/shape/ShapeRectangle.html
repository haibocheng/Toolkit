draw()method
override public function draw():void

This method is invoked everytime arrange is called. arrange() will clear the graphics object everytime. The correct width and height is then calculated based on the usecases outlined below: o No width + height explicitely set so use decorated width + height values o Previously decorated width + height values set and decorated arrange() is called again due to children updating their width + height properties. o Explicitely set width + height properties thus ignore decorated width + height
