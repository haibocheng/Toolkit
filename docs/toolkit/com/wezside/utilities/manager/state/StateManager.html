stateKeyproperty 
stateKey:String

Contains the concatenated keys of all states which bits are true, i.e. 1 and not 0. Non-reserved states are mutually exclusive whereas reserved states is allowed to co-exist with both reserved and non-reserved states. This String key will reflect this logic.
