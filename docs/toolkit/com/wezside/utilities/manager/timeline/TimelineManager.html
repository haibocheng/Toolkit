addElement()method
public function addElement(id:String, target:MovieClip, delay:int = 0, endFrame:int = -1, autoVisible:Boolean = false, childPolicy:String):void

Add a new timeline element to the manager. This method will create a new TimelineInstance element.

Parameters

id:String — A unique ID fo referencing a specific TimelineInstance
 
target:MovieClip — The Target embedded Movieclip
 
delay:int (default = 0) — A start delay before the TimelineInstance starts playing
 
endFrame:int (default = -1) — An end frame number for when the TimelineEvent.COMPLETE event is dispatch. Leave as -1 to use the totalFrames of the MovieClip
 
autoVisible:Boolean (default = false) — Toggels visibility on and off at start and end of playback
 
childPolicy:String (default = NaN) — A policy defining if gotoAndStop(1) is called on all children at start and end of playback. Default is root only.
