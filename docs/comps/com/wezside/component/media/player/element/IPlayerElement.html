displayWidthproperty 
displayWidth:int

The displayWidth property is set by the Player class upon receiving the Meta info for the specific media element. In order to accurately determine the autosize width of any IPlayerControl elements (IControlElement) we divide the collection by autoSize is true and false respectively.

The algorithm that achieves this works as follows: 1. Determine what the remaining width is after all non-autosize element's width were deducted 2. Set each autosize true element width = to the remainder width calculated above divided by the total autosize true elements so each element get an equal share of the remainder width
